#  Tour de Hanoï - Solution Itérative avec Visualisation

> Une implémentation interactive du célèbre problème de la Tour de Hanoï utilisant un algorithme itératif avec visualisation en temps réel dans la console.
> **Groupe 6, L1B** : Muckaya, Njaratiana, Maelle, Fitia, Shanty  
> **Sujet n°6** : Implémentation itérative de la Tour de Hanoï avec affichage  


##  Présentation

Ce projet implémente le problème classique de la **Tour de Hanoï** en utilisant une approche **itérative** (sans récursion) avec une magnifique visualisation dans la console. Regardez la solution se dérouler étape par étape avec des mouvements animés !

###  Fonctionnalités

-  **Algorithme Itératif** - Pas de récursion, pure logique itérative
-  **Visualisation en Temps Réel** - Affichage graphique avec des disques proportionnels
-  **Mode Pas-à-Pas** - Suivez chaque mouvement avec retour visuel
-  **Validation Automatique** - Vérifie que la solution est correcte
-  **Transitions Animées** - Pauses fluides entre les mouvements
-  **Double Implémentation** - Disponible en Python et en C
-  **Entrée Interactive** - Choisissez le nombre de disques (1-8)
